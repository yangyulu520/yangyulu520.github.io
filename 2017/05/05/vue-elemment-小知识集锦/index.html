vue+elemment-小知识集锦

前端
  • 前端

跨域时，vue请求后端，不会携带cookie，从而导致每次请求，都无法携带session来进行会话认证，导致一系列问题。如果出现认证方便的问题，看请求头中是否携带了cookie导致的问题。

项目添加vuex

项目添加elementUI:cnpm i element-ui -S

跨域时，更新资源的请求会二次提交，注意options请求的处理

npm install -g cnpm –registry=https://registry.npm.taobao.org

安装cnpm,用cnpm下载东西有时会快一些

main.js里加载的第三方组件，页面中访问，需要用this.$xxx的原因

main.js里面，在加载外第三方组件后，同时把组件挂载到了Vue的实例下面了，所以页面组件中使用，需要从Vue实例中取。而其他地方引用组件后，则使用组件名来引用。

main.js中Vue.use(组件)和newVue(组件)

Vue.use(组件)是使用该组件，相当于先注册，然后使用。而newVue(组件)是实例化该组件，相当于启动。
在main.js里加载组件需要先use该组件，然后再挂载到Vue实例下。也可以把use放到具体组件js下，main.js里直接加载该js就行。

axios默认发送的是json请求
