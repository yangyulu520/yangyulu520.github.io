mysql主从数据库搭建-基础环境

数据库
  • mysql

基础环境

环境

centos6.4+mysql5.6+VMWare10

文件存放

云盘MySQL数据库目录

rz文件上传功能安装

命令：yum install lrzsz

mysql主从库数据库的安装

  1. 把rpm包上传到linux系统。
  2. 安装rpm到linux系统。
    yum localinstall mysql-community-release-el6-5.noarch.rpm
  3. 使用yum进行在线安装，需要联网。
    yum install mysql-community-server
  4. 启动mysql服务
    service mysqld start
  5. 设置mysql root用户的密码
    /usr/bin/mysqladmin -u root password ‘new-password’
  6. 授权远程登录
    GRANT ALL PRIVILEGES ON . TO ‘root’@’%’ IDENTIFIED BY ‘yangyl’ WITH GRANT OPTION;
