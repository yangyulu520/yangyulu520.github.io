java注解

后端
  • java

类及类上声明说明

说明：

  1. @target:表示该注解可以使用的范围，这里标明可以在方法和类级别上使用该注解。具体范围如下：
    @Target(ElementType.TYPE) //接口、类、枚举、注解
    @Target(ElementType.FIELD)//字段、枚举的常量
    @Target(ElementType.METHOD)//方法
    @Target(ElementType.PARAMETER)//方法参数
    @Target(ElementType.CONSTRUCTOR) //构造函数
    @Target(ElementType.LOCAL_VARIABLE)//局部变量
    @Target(ElementType.ANNOTATION_TYPE)//注解
    @Target(ElementType.PACKAGE) ///包

  2. @Retention:表示该类型的注解会被保留到哪个阶段，具体阶段如下：
    @RetentionPolicy.SOURCE —— 这种类型的Annotations只在源代码级别保留,编译时就会被忽略
    @RetentionPolicy.CLASS —— 这种类型的Annotations编译时被保留,在class文件中存在,但JVM将会忽略
    @RetentionPolicy.RUNTIME —— 这种类型的Annotations将被JVM保留,所以他们能在运行时被JVM或其他使

  3. @Inherited：说明子类可以继承父类中的该注解

  4. @Documented：说明该注解将会被包含在Javadoc中

  5. @interface:声明类为注解类
